# 844. 比较含退格的字符串
[题目链接](https://leetcode.cn/problems/backspace-string-compare/description/)

---
## 思路
>双指针

### ***C++***
```CPP
class Solution {
public:
    bool backspaceCompare(string s, string t) {
        int sSkip = 0, tSkip = 0;
        int ps = s.size() - 1, pt = t.size() - 1;
        for (;;)
        {
            //处理s中的'#'
            while (ps >= 0)
            {
                if (s[ps] == '#')
                    sSkip++;
                else if (sSkip > 0)
                    sSkip--;
                else 
                    break;
                ps--;
            }
            //处理t中的'#'
            while (pt >= 0)
            {
                if (t[pt] == '#')
                    tSkip++;
                else if (tSkip > 0)
                    tSkip--;
                else 
                    break;
                pt--;
            }
            //比较
            if (pt < 0 && ps < 0)
                return true;
            if (pt < 0 || ps < 0)
                return false;
            if (s[ps] != t[pt])
                return false;
            pt--;
            ps--;
        }
        return true;
    }
};
```
