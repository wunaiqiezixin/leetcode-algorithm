# 34. 在排序数组中查找元素的第一个和最后一个位置

[力扣题目链接](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/description/)

给你一个按照非递减顺序排列的整数数组 nums，和一个目标值 target。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

你必须设计并实现时间复杂度为 O(log n) 的算法解决此问题。



示例 1：

输入：nums = [5,7,7,8,8,10], target = 8

输出：[3,4]

示例 2：

输入：nums = [5,7,7,8,8,10], target = 6

输出：[-1,-1]

示例 3：

输入：nums = [], target = 0

输出：[-1,-1]


### 提示：

* 0 <= nums.length <= 105
* -109 <= nums[i] <= 109
* nums 是一个非递减数组
* -109 <= target <= 109

## 思路：

* 二分查找两次

* 第一次查找左边界，第二次查找有边界

* 以查找左边界为例：每当 nums[mid] = target 时，记录左边界的下标为 best = mid ,但我们不能保证这个值是最左边的，因此，再令 right = mid - 1 ,查找best左侧的区间，确保best为满足条件的最左边的元素的下标

### ***C：***
```C
/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
int getLeft(int* nums,int numsSize,int target)
{
    int left = 0,right = numsSize - 1,best = -1;
    int mid;
    while (left <= right)
    {
        mid = left + (right - left) / 2;
        if (nums[mid] < target)
        {
            left = mid + 1;
        }
        else if (nums[mid] > target)
        {
            right = mid - 1;
        }
        else
        {
            best = mid;
            right = mid - 1;
        }
    }
    return best;
}
int getRight(int* nums,int numsSize,int target)
{
    int left = 0,right = numsSize - 1,best = -1;
    int mid;
    while (left <= right)
    {
        mid = left + (right - left) / 2;
        if (nums[mid] < target)
        {
            left = mid + 1;
        }
        else if (nums[mid] > target)
        {
            right = mid - 1;
        }
        else
        {
            best = mid;
            left = mid + 1;
        }
    }
    return best;
}
int* searchRange(int* nums, int numsSize, int target, int* returnSize) {
    *returnSize = 2;
    int leftBoundary = getLeft(nums,numsSize,target);
    int rightBoundary = getRight(nums,numsSize,target);
    int* result = (int*)malloc(sizeof(int) * 2);
    result[0] = leftBoundary;
    result[1] = rightBoundary;
    return result;
}
```

### ***C++***
```CPP
class Solution {
int lower_bound(const vector<int>& nums, int target)
{
    int left = 0, right = (int)nums.size() - 1;
    while (left <= right)
    {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target)
            left = mid + 1;
        else
            right = mid - 1;
    }
    return left;
}
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        int ans_left = lower_bound(nums, target);
        if (ans_left >= nums.size() || nums[ans_left] != target)
            return {-1, -1};
        int ans_right = lower_bound(nums, target + 1) - 1;
        return {ans_left, ans_right};
    }
};
```
## 思路2：lower_bound()

### 循环不变量：
**左闭右闭**
`int left = 0, right = nums.size() - 1;`
- 下标小于 left 的元素的值都小于 target ( left 是第一个大于等于 target 的下标)
- 下标大于 right 的元素的值都大于等于 target

### ***C++***

```CPP
class Solution {
private:
    int lower_bound(const vector<int>& nums, int target) {
        int n = nums.size();
        // 循环不变量：
        int left = 0;       // 下标小于 left的元素的值 <  target
        int right = n - 1;  // 下标大于right的元素的值 >= target
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] >= target) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return left;
    }
public:
    vector<int> searchRange(vector<int>& nums, int target) {
        int left_ans = lower_bound(nums, target);
        if (left_ans >= nums.size() || nums[left_ans] != target) { // 不存在target的情况
            return {-1, -1};
        }
        int right_ans = lower_bound(nums, target + 1) - 1; // 最后一个等于target的下标
        return {left_ans, right_ans};
    }
};
```
