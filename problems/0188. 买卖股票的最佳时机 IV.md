# 188. 买卖股票的最佳时机 IV
[力扣题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock-iv/description/)

---
## 思路
>三维dp
1. dp数组的含义：  
`dp[i][0][j]` 第i天第j次买入股票的最大利润  
`dp[i][1][j]` 第i天第j次卖出股票的最大利润  
2. dp数组的初始化  
`dp[0][0][j] = -prices[0]` ：第0天，不论是第几次买入股票，成本都是`prices[0]`(在同一天重复买入卖出)  
`dp[0][1][j] = 0` ：第0天，不论是第几次卖出股票，利润都是0，因为重复的买入卖出，买卖了个寂寞  
3. 动态转移方程  
`dp[i][0][j] = max( dp[i - 1][0][j], dp[i - 1][1][j - 1] - prices[i] )`  
`dp[i][1][j] = max( dp[i - 1][1][j], dp[i - 1][0][j] + prices[i] )`  
4. 遍历顺序  
i - 1 -> i  
j - 1 -> j  
遍历顺序任意  

### ***C++***
```CPP
class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        int len = prices.size();
        /*
            dp[i][0][j]:第i天第j次买入股票
            dp[i][1][j]:第i天第j次卖出股票
        */
        vector<vector<vector<int> > > dp(len, vector<vector<int> >(2, vector<int>(k + 1, 0)));
        int i, j;
        for (j = 0; j <= k; ++j)
            dp[0][0][j] = -prices[0];
        for (i = 1; i < len; ++i)
        {
            for (j = 1; j <= k; ++j)
            {
                dp[i][0][j] = max(
                    dp[i - 1][0][j],
                    dp[i - 1][1][j - 1] - prices[i]
                );
                dp[i][1][j] = max(
                    dp[i - 1][1][j],
                    dp[i - 1][0][j] + prices[i]
                );
            }
        }
        int maxProfit = 0;
        for (j = 0; j <= k; ++j)
        {
            if (dp[len - 1][1][j] > maxProfit)
                maxProfit = dp[len - 1][1][j];
        }
        return maxProfit;
    }
};
```

>优化空间
```CPP
class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        int len = prices.size();
        /*
            dp[0][j]:第i天第j次买入股票
            dp[1][j]:第i天第j次卖出股票
        */
        vector<vector<int> > dp(2, vector<int>(k + 1, 0));
        int i, j;
        for (j = 0; j <= k; ++j)
            dp[0][j] = -prices[0];
        for (i = 1; i < len; ++i)
        {
            for (j = 1; j <= k; ++j)
            {
                dp[0][j] = max(
                    dp[0][j],
                    dp[1][j - 1] - prices[i]
                );
                dp[1][j] = max(
                    dp[1][j],
                    dp[0][j] + prices[i]
                );
            }
        }
        int maxProfit = 0;
        for (j = 0; j <= k; ++j)
        {
            if (dp[1][j] > maxProfit)
                maxProfit = dp[1][j];
        }
        return maxProfit;
    }
};
```

## 二、核心思路：2k+1 状态合并法
本解法最巧妙的地方在于：用一个整数维度同时表示两个信息 —— 已完成的交易次数和当前是否持有股票，将原本三维的 DP 状态 dp[i][j][0/1] 压缩为二维 dp[i][j]，极大简化了代码。  
**状态定义**  
我们将每一笔完整的交易拆分为买入和卖出两个连续动作，用 2k+1 个状态来覆盖所有可能的交易情况：  

|状态编号 j|含义	|对应三维 DP状态|
|---|---|---|
|0	|初始状态：0 笔交易，不持有股票|	dp[i][0][0]|
|1	|第 1 次买入后：持有股票，正在进行第 1 笔交易|dp[i][1][1]|
|2	|第 1 次卖出后：不持有股票，已完成 1 笔交易|dp[i][1][0]|
|3	|第 2 次买入后：持有股票，正在进行第 2 笔交易|dp[i][2][1]|
|4|第 2 次卖出后：不持有股票，已完成 2 笔交易|dp[i][2][0]|
|...|	...|	...|
|2m|第 m 次卖出后：不持有股票，已完成 m 笔交易|dp[i][m][0]|
|2m+1|第 m+1 次买入后：持有股票，正在进行第 m+1 笔交易|dp[i][m+1][1]|
|...|...|...|
|2k|第 k 次卖出后：不持有股票，已完成 k 笔交易|dp[i][k][0]|

**核心规律总结：**  
- 偶数 j：不持有股票，已完成 j/2 笔交易
- 奇数 j：持有股票，正在进行第 (j+1)/2 笔交易




```CPP
class Solution {
public:
    int maxProfit(int k, vector<int>& prices) {
        int n = prices.size();
        if (k >= n / 2) {
            int max_profit = 0;
            for (int i = 1; i < n; ++i) {
                int diff = prices[i] - prices[i - 1];
                if (diff > 0) {
                    max_profit += diff;
                }
            }
            return max_profit;
        }
        vector<vector<int>> dp(n, vector<int>(2 * k + 1, 0));
        for (int j = 1; j < 2 * k; j += 2) {
            dp[0][j] = -prices[0];
        }
        for (int i = 1; i < n; ++i) {
            for (int j = 0; j < 2 * k; j += 2) {
                dp[i][j + 1] = max(dp[i - 1][j + 1], dp[i - 1][j] - prices[i]);//第i天，持有股票
                dp[i][j + 2] = max(dp[i - 1][j + 2], dp[i - 1][j + 1] + prices[i]);//第i天，不持有股票
            }
        }
        return dp[n - 1][2 * k];
    }
};
```
