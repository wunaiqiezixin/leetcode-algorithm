## [蓝桥杯2026年第十七届省赛真题-蓝桥竞技](https://www.dotcpp.com/oj/problem3352.html)

---

> 这是一道数学结论题

## 思路
**结论：**
1. n 个数的总和 total 必须是 5 的倍数
2. 每一个数的值必须小于组数 total / 5

## 写代码

唯一要注意的点: total 会溢出，用 long long

## ***C++***

```C++
#include <iostream>
using namespace std;
using ll = long long;

void solve() {
    int n;
    cin >> n;
    ll total = 0;
    ll max_val = 0;
    for (int i = 0; i < n; ++i) {
        ll val;
        cin >> val;
        total += val;
        if (val > max_val) max_val = val;
    }
    if (total % 5) {
        cout << "F";
    } else if (max_val > (total / 5)) {
        cout << "F";
    } else {
        cout << "T";
    }
    cout << '\n';
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int t;
    cin >> t;
    while (t--) {
        solve();
    }
    return 0;
}
```
