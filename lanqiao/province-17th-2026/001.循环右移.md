## [蓝桥杯2026年第十七届省赛真题-循环右移](https://www.dotcpp.com/oj/problem3351.html)
---
>这是一道送命题

## 题目重述
给定一个有 n 个元素的数组 a，其中 X<=ai<=Y, 且对于任意的子数组进行一次循环右移后，数组保持不变。求满足条件的数组的个数。

## 思路
- 先考虑长度为2的数组，显然两个元素必须相等。
- 再考虑长度为3的数组，三个元素也必须相等。
- 很轻易地就能推出，数组中的相邻元素两两相等，所以整个数组中的元素必然相等。

**于是数组长度 n 没有任何的用处，满足条件的数组个数只和 X、Y 有关：**
1. 当 X > Y 时：数组个数为 0
2. 当 X <= Y 时：数组个数为 Y - X + 1

## 写代码

这里主要考虑溢出的问题，其余没有任何难度。

>对于 100% 的评测用例，$1 ≤ T ≤ 10^3，1 ≤ N ≤ 10^{18}，1 ≤ X, Y ≤ 10^{18}$。

最大的数为 $10^{18}$ , 用 uint64_t 存即可

|类型|字节数|最小值|最大值|直观量级|
|---|---|---|---|---|
|int32_t|4|-2147483648|2147483647|正负约 21 亿|
|uint32_t|4|0|4294967295|约 42 亿|
|int64_t|8|-9223372036854775808|	9223372036854775807|正负约 900 亿亿|
uint64_t|8|0|18446744073709551615|约 1800 亿亿|

## ***C++***
```CPP
#include <iostream>
#include <cstdint>
using namespace std;

void solve() {
    uint64_t N, X, Y;
    cin >> N >> X >> Y;
    if (X > Y) {
        cout << 0 << '\n';
    } else {
        cout << Y - X + 1 << '\n';
    }
}

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int t;
    cin >> t;
    while (t--) {
        solve();
    }
    return 0;
}
```
