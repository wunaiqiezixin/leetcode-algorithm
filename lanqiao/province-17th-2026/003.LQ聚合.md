## [蓝桥杯2026年第十七届省赛真题-LQ聚合](https://www.dotcpp.com/oj/problem3353.html)

---

## 题目重述
给定一个长度为 n 的仅包含字符 L、Q 和 ? 的字符串。
将全部问号更换为 L 或 Q ，求最大 "LQ聚合"。

## 思路
**结论：**
>**如果将一个问号变为了 L ，那么它的左边的问号都要变为 L ; 如果将一个问号变为了 Q ，那么它右边的问号都要变为 Q 。**

- 于是我们只要枚举每个分界点 k 即可！
- 将前 k 个问号变为 L ，后 m-k 个问号变为 Q 。


**"LQ聚合数" 只由以下四部分产生：**
1. 原有的 L 和 Q 的贡献(每个 Q 前面的 L 的个数的和): `fixed`
2. 将每个问号变为 L 产生的贡献(每个问号后面 Q 的个数): `vector<ll> vL`
3. 将每个问号变为 Q 产生的贡献(每个问号前面 L 的个数): `vector<ll> vQ`
4. 将所有问号都变为 L 或 Q 后，互相产生的贡献: 变为 L 的问号的个数 * 变为 Q 的问号的个数


## 写代码

1. 从左往右遍历字符串：
用一个变量 `cnt_L` 表示已经遍历到的 L 的数量。
当遇到 L 时，`cnt_L++`
当遇到 Q 时，`fixed += cnt_L` (原有的 L 和 Q 产生的贡献)
当遇到 ？ 时，`vQ.push_back(cnt_L)` (将每个问号变为 Q 产生的贡献)
一次遍历后可以得到问号的总数为: `m = vQ.size()`
2. 从右往左遍历字符串:
用一个变量 `cnt_Q` 表示已经遍历到的 Q 的数量。
用一个变量 `index = m - 1` 表示 `vL` 的下标。
当遇到 Q 时，`cnt_Q++`
当遇到 ? 时，`vL[idex--] = cnt_Q` (每个问号变为 L 产生的贡献)

3. 统计前缀/后缀和:
`vector<ll> pre_vL(m + 1, 0)` : `0~k-1` 个问号变为 L 产生的贡献
`vector<ll> suf_vQ(m + 1, 0)` : `k~m-1` 个问号变为 Q 产生的贡献

4. 枚举分界点 k 取最大值:
将 0~k-1 个问号变为 L
将 k~m-1 个问号变为 Q

## ***C++***
> 我已尽力，但是以下代码只有 69 分！！！

```C++
#include <iostream>
#include <vector>
#include <string>
using namespace std;
using ll = long long;
int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    string s;
    cin >> n >> s;

    ll fixed = 0; // 原有的 L Q 产生的贡献

    int cnt_L = 0;
    vector<ll> vQ; // 将问号变为 Q 产生的贡献
    for (char ch : s) {
        if (ch == 'L') cnt_L++;
        else if (ch == 'Q') fixed += cnt_L;
        else vQ.push_back(cnt_L);
    }
    int m = vQ.size();

    int idx = m - 1;
    int cnt_Q = 0;
    vector<ll> vL(m); // 将问号变为 L 产生的贡献
    for (int i = n - 1; i >= 0; i--) {
        char ch = s[i];
        if (ch == 'Q') cnt_Q++;
        else if (ch == '?') vL[idx--] = cnt_Q;
    }

    vector<ll> pre_vL(m + 1, 0); //pre_vL[k]: 0~k-1 的问号变为了 L 产生的贡献
    vector<ll> suf_vQ(m + 1, 0); //suf_vQ[k]: m-1~k 的问号变为了 Q 产生的贡献

    for (int i = 0; i < m; i++) {
        pre_vL[i + 1] = pre_vL[i] + vL[i];
    }
    for (int i = m - 1; i >= 0; i--) {
        suf_vQ[i] = suf_vQ[i + 1] + vQ[i];
    }

    ll max_val = 0;
    for (int k = 0; k <= m; k++) {
        // 0~k-1变为L; k~m-1 变为Q
        ll cur = 1LL * k * (m - k) + pre_vL[k] + suf_vQ[k];
        if (cur > max_val) max_val = cur;
    }

    cout << max_val + fixed << '\n';

    return 0;

}
```
