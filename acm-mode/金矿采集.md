## 题目核心重述
在 n×m 的网格中，飞船从起点 S 出发，可上下左右四向移动，障碍物不可通行。网格中有 k 个金矿，每个金矿初始价值 Xi，价值衰减规则如下：  
- 当已采集 a 个金矿时，每移动一步，所有未采集的金矿价值减少 a 点，最低降至 1。  
- 到达金矿即可采集，采集顺序任意，求最终能获得的最大总价值。  
  
**数据范围：** 1≤n,m≤50，1≤k≤5，1≤Xi≤1000，保证所有金矿可达。  

## 核心思路总览
>本题的突破口是金矿数量 k 极小（最多 5 个），因此可以采用「网格最短路预处理 + 全排列枚举采集顺序」的朴素策略，时间复杂度完全可控。  

**整体分为三步：**
1. 抽象关键点：将 起点 和 k 个金矿视为 k+1 个关键节点，通过 BFS 预处理出所有关键点两两之间的最短路径长度。  
2. 推导衰减公式：根据移动阶段与已采集数量的对应关系，推导出每个金矿的累计扣减计算方式。  
3. 全排列枚举：遍历所有 k! 种采集顺序，按规则计算总价值，取最大值。  

## 详细原理推导
1. **最短路的最优性证明**  
任意两个关键点之间，走最短路径一定是最优选择：  
- 移动步数越少，累计价值扣减越少，总价值越高。
- 绕路只会增加移动步数，导致扣减更多，不可能得到更优结果。
- 因此我们只需要预处理关键点之间的最短距离，无需考虑网格中的具体路径。
- 由于网格是无权图（每步代价为 1），使用广度优先搜索（BFS） 可以在线性时间内求出单源最短路径。  

2. **价值衰减规则的数学推导**
这是本题最容易出错的部分，我们按阶段拆解：  
- 初始状态：已采集数量 a=0，所有金矿价值为初始值。
- 阶段 1：从起点出发，前往第 1 个金矿  
此阶段 a=0，每步扣减 0，所有金矿价值不变。到达后采集第 1 个金矿，a 变为 1。
- 阶段 2：从第 1 个金矿前往第 2 个金矿  
此阶段 a=1，每步扣减 1，剩余未采集的金矿每走一步都扣 1。到达后采集第 2 个金矿，a 变为 2。  
- 阶段 i：从第 i-1 个金矿前往第 i 个金矿  
此阶段 a=i−1，剩余未采集的金矿每走一步扣 i−1。  

**结论：** 对于第 i 个被采集的金矿（i 从 1 开始），它的累计扣减量 = 所有前往它之前的各阶段「步数 × 当时的 a 值」之和。  
  
>我们可以用一个变量 `cum_drop` 记录累计扣减，每走完一段路就累加本次的扣减，到达金矿时直接用初始值减去累计扣减即可。  

3. **全排列的合理性**  
- k≤5，全排列总数仅为 5!=120 种，计算量极小。  
- 最优解一定是采集全部金矿：每个金矿最低价值为 1，采集至少获得 1 点收益，不采集收益为 0，因此全采一定优于部分采集。  


## 算法执行步骤
**步骤 1：**  
- 提取关键点坐标遍历网格，记录起点 S 和编号 1~k 的金矿的坐标，共 k+1 个关键点。  

**步骤 2：**  
- BFS 预处理距离矩阵对每个关键点执行一次 BFS，计算其到网格中所有格子的最短距离，提取其他关键点的距离，得到 `(k+1)×(k+1)` 的距离矩阵 `dist[i][j]`，表示第 i 个关键点到第 j 个关键点的最短步数。

**步骤 3：**  
- 枚举所有采集顺序生成 1~k 的全排列，对每种采集顺序：  
初始化：当前位置为起点，累计扣减为 0，总价值为 0，已采集数量 a=0。  
按顺序前往每个金矿：累加本段扣减：`cum_drop += a * 本段距离`  
计算当前金矿价值：`max(1, X[金矿编号] - cum_drop)`，加入总价值  
更新当前位置，已采集数量 a+1  
更新全局最大总价值。    

**步骤 4：**  
- 输出最大值所有排列遍历完成后，输出最大总价值。  


## 复杂度分析
- **时间复杂度：`O(k⋅n⋅m+k!⋅k)`**  
BFS 部分：共 k+1 次 BFS，每次遍历全网格 O(nm)，总规模约 6×50×50=15000 次操作。  
全排列部分：共 k! 种顺序，每种遍历 k 个金矿，k=5 时仅 120×5=600 次操作。  

- **空间复杂度：O(n⋅m+k 2 )** ，主要为 BFS 的距离数组和关键点距离矩阵。  

整体计算量极小，远低于时间限制。  


## 代码实现  
### ***C++***  

```CPP
#include <iostream>
#include <vector>
#include <string>
#include <queue>
#include <algorithm>
using namespace std;

int main() {
    // 关闭同步加速输入
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int n, m, k;
    cin >> n >> m >> k;

    vector<string> grid(n);
    for (int i = 0; i < n; ++i) {
        cin >> grid[i];
    }

    vector<int> X(k + 1); // X[1]~X[k] 对应编号1~k的金矿初始价值
    for (int i = 1; i <= k; ++i) {
        cin >> X[i];
    }

    // pos[0]存起点S坐标，pos[1]~pos[k]存对应编号金矿的坐标
    vector<pair<int, int>> pos(k + 1, {-1, -1});
    for (int r = 0; r < n; ++r) {
        for (int c = 0; c < m; ++c) {
            char ch = grid[r][c];
            if (ch == 'S') {
                pos[0] = {r, c};
            } else if (ch >= '1' && ch <= '5') {
                int id = ch - '0';
                if (id >= 1 && id <= k) { // 防止编号超出k的范围
                    pos[id] = {r, c};
                }
            }
        }
    }

    // 四个方向：上下左右
    int dr[] = {-1, 1, 0, 0};
    int dc[] = {0, 0, -1, 1};

    // dist[i][j]：第i个关键点到第j个关键点的最短距离
    vector<vector<int>> dist(k + 1, vector<int>(k + 1, 0));

    // 对每个关键点做一次BFS
    for (int i = 0; i <= k; ++i) {
        if (pos[i].first == -1) continue;

        vector<vector<int>> d(n, vector<int>(m, -1)); // -1表示未访问
        queue<pair<int, int>> q;

        int sr = pos[i].first;
        int sc = pos[i].second;
        d[sr][sc] = 0;
        q.push({sr, sc});

        while (!q.empty()) {
            auto [r, c] = q.front();
            q.pop();

            for (int dir = 0; dir < 4; ++dir) {
                int nr = r + dr[dir];
                int nc = c + dc[dir];
                // 边界合法、不是障碍物、未访问过
                if (nr >= 0 && nr < n && nc >= 0 && nc < m
                    && grid[nr][nc] != '#'
                    && d[nr][nc] == -1) {
                    d[nr][nc] = d[r][c] + 1;
                    q.push({nr, nc});
                }
            }
        }

        // 提取所有关键点的距离
        for (int j = 0; j <= k; ++j) {
            if (pos[j].first != -1) {
                dist[i][j] = d[pos[j].first][pos[j].second];
            }
        }
    }

    // 生成初始排列：1, 2, ..., k
    vector<int> perm;
    for (int i = 1; i <= k; ++i) {
        perm.push_back(i);
    }

    long long max_total = 0;

    // 枚举所有全排列
    do {
        int cur = 0;          // 当前所在关键点编号，初始为起点0
        long long cum_drop = 0; // 累计扣减值
        long long total = 0;  // 当前排列的总价值

        for (int i = 0; i < k; ++i) {
            int nxt = perm[i];
            int d = dist[cur][nxt];

            // 累加本段路程的扣减：已采集i个，每步扣i点
            cum_drop += 1LL * i * d;

            // 计算当前金矿的实际价值，最低为1
            int val = max(1, X[nxt] - (int)cum_drop);
            total += val;

            // 更新状态
            cur = nxt;
        }

        max_total = max(max_total, total);
    } while (next_permutation(perm.begin(), perm.end()));

    cout << max_total << "\n";
    return 0;
}
```
